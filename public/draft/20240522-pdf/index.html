PDF to text conversion#

See https://superuser.com/questions/286961/pdf-to-text-convertor. It is part of xpdf, and can be installed using:

brew install xpdf

Another option according to above is poppler:

brew install poppler

Poppler is a library, not a command line utility.

I haven’t tried either so far.

xpdf#

xpdf installs the followint command line tools:

  • xpdf: PDF viewer (click for a screenshot)
  • pdftotext: converts PDF to text
  • pdftops: converts PDF to PostScript
  • pdftoppm: converts PDF pages to netpbm (PPM/PGM/PBM) image files
  • pdftopng: converts PDF pages to PNG image files
  • pdftohtml: converts PDF to HTML
  • pdfinfo: extracts PDF metadata
  • pdfimages: extracts raw images from PDF files
  • pdffonts: lists fonts used in PDF files
  • pdfdetach: extracts attached files from PDF files
